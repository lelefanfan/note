可以使用 literal 标签来防止模板标签被解析，主要用于防止javascript标签被解析。例如：


```html
<script>
<!-- {literal} -->
function selectPage(sel)
{
    sel.form.submit();
}
<!-- {/literal} -->
</script>
```
