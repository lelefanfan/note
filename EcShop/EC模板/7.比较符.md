#比较符

[TOC]

在EC模板里支持两种比较符号：

| 功能 | 符号 | 字符 |
| :-------- | :--------| :-------- |
| 等于号 |  == | eq |
| 不等于号 |  != | ne/neq |
| 大于号 | > | gt |
| 大于等于号 | >= | ge/gte |
| 小于号 | < | lt |
| 小于等于号 | <= | le/lte |
| 并且 | && | and |
| 或 | &#124;&#124; | or |
| 非 | ! | not |
| 求模 | % | mod |
