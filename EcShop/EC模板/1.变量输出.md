#变量输出

[TOC]

在模板中输出变量的方法很简单，例如，在控制页中我们给模板变量赋值：

```php
// 当前 index.php 页面
$name = 'LinWu';
$smarty->assign('name',$name);
$smarty->display('index.dwt');
```

然后就可以在index.dwt模板中使用：

```html
Hello,{$name}！
```

这样，运行的时候就会在模板中显示： `Hello,LinWu！`
